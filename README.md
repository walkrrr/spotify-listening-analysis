# Spotify Listening Data Analysis

Analysis of personal Spotify streaming history using Python and Jupyter Notebook.

## Project Highlights
• Listening heatmap by day and hour  
• Top artists and songs  
• Listening trends over time  
• Music vs podcast analysis

## Tools
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Visualizations
- Listening heatmap
- Top artists
- Listening activity by hour
- Monthly listening trend
- Music vs podcast distribution
- Top songs
