# Spotify Listening Analysis

This project analyzes Spotify streaming history to understand listening behavior and patterns.

## Tools
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Visualizations
- Listening heatmap
- Top artists
- Listening activity by hour
- Monthly listening trend
- Music vs podcast distribution
- Top songs
